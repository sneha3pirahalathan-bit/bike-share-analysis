# Dataset Description

This project uses Divvy Bike Share trip data from November 2020 to November 2021.

## Dataset Overview

The original dataset contains ride-level information including:

* Ride ID
* Rideable Bike Type
* Start Time
* End Time
* Start Station
* End Station
* Rider Type (Member or Casual)

## Data Source

The dataset is publicly available from Divvy Bike Share:

https://divvy-tripdata.s3.amazonaws.com/index.html

## Note

Due to GitHub file size limitations, the complete dataset is not stored in this repository.

The original data can be downloaded directly from the source above.
